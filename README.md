# PyGame Autoplayer
### why scroll when you can have the videos play themselves
INSTRUCTIONS:
- Install every dependency in `main.py` (under LIBRARIES)
- Adjust preferences in `main.py` (under CUSTOMIZATION)
- Put all the videos you want to play in the `vids` folder.
- Run `update_txtfile.py`  

Then you can run `main.py` to watch all the videos you wanna,
> [!NOTE]
> Because PyGame is just not optimized, there could be desyncs. But I don't care
